GIT: 
    Сначала регистрируемся на https://github.com и кидаем ники чтобы добавил в collaborators
    Каждый добавляет следующим образом любой файлик с любым содержимом для теста
    0. Полезная ссылка https://githowto.com/ru
       Полезная команда git status
    1. Создаем пустую папку RTS
        mkdir RTS
    2. Заходим
        cd RTS
    3. Мутим репу в папке
        git init
    4. Связываем этот репозиторий с моим
        git remote add origin https://github.com/zhog96/RTS.git
    5. Скачиваем мой репозиторий в свой (иначе при попытке положить на серв ничего не получится)
        git pull origin master
    6. Теперь создаем отдельную ветку с любым названием, не лепим в master вообще, никто
	git checkout -b <Имя ветки>
       Мы автоматически оказались на новосозданной ветке
    7. Тут делаем что считаем нужным
    8. Чтобы заиндексировать изменения
        git add .
    9. И пушим
        git push -u origin <Имя ветки( _НЕ_ master)>
    10. Далее я сливаю ветку в master
    При дальнейшей работе повторяем цикл 5 - 9
    Кто будет добавлять файлики ручками рискует их лишиться :)
